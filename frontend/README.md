# `frontend/` — the TypeScript console (NOT `web/`)

The first-class UX surface — where governed autonomy becomes *usable*. It consumes the generated
OpenAPI (P1-11), never a second contract ([O] INV-15; [R] paradigm-rule 4).

First read-only views (P1-10): the **ActionManifest timeline** (predicted → approved → executed →
verified as one visual chain), the **tamper-evident ledger** view, an **explainability** panel, and
the **approval console** shell (the human-circuit-breaker surface — inert while mutation is OFF, lit
up in Phase 2). **Autonomy-band + kill-switch controls live here** (API/RBAC-driven, audited on
change), never on host-local sentinel files. Multi-tenant admin is a first-class dimension of every
view.

**Status:** scaffold is **P1-10**. Framework (React/Svelte) deferred (ADR-0006).
