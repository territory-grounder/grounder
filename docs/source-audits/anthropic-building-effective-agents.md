# Anthropic — Building Effective Agents

> **[O] audit-overlay · TG-38 source benchmark.** Category: **Anthropic** · TG adherence: **4.1 / 5** · Practices audited: **10** (implemented 6 · partial 3 · gap 0 · N/A-by-design 1).

## Verdict

TG faithfully implements most of Anthropic's pattern progression and does so with governance the source only gestures at. Augmented-LLM, prompt-chaining (the Temporal Runner chain), autonomous-agent (deliberately clamped to propose-only), and especially the evaluator-optimizer (a statistically-gated skill flywheel with Welch t-test, offline admission gate, and auto-demoting regression watch) are strong-to-exceeding, and stopping conditions are comprehensive. Orchestrator-workers is correctly N/A (single native agent + deterministic orchestration, INV-08). The real shortfalls: routing is decided but not yet branching execution (no simplicity payoff delivered), true parallelization is absent (substituted by independent mechanical-verify + async judge), and 'parallelized guardrails' is honored via deterministic separate gates rather than a parallel LLM screen.

## Top gaps

- Wire execclass routing to branch execution paths so cheap incidents skip the full lifecycle
- Add a parallel LLM content/harm screen (or an explicit per-session token-cost cap) for high-severity proposals
- Only-if-measured: add multi-perspective (voting/sectioning) diagnosis verification on critical incidents
